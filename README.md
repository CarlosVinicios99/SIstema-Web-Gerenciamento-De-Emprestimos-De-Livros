# Sistema Web - Gerenciamento De Empréstimos De Livros

<p>
  Neste projeto foi desenvolvido um Sistema de Gerenciamento De Empréstimos De Livros com base em uma biblioteca acadêmica que possibilita que usuários possam disponibilizar e emprestar livros entre si.<br>
<br> Este projeto foi desenvolvido durante o TCC para o Curso De Ciência Da Computação pela Universidade Federal Fluminense<br>
Desenvolvido em dupla com meu colega <a href="https://github.com/ColdmaterL">Lucas Mendonça</a>
</p>
<p>
  Mais detalhes sobre o projeto podem ser encontrados na nossa dissertação.
</p>

## Navegação e Principais Funcionalidades
<p>
  Existem duas perspectivas no sistema, a perspectiva de Bibliotecário e a perspectiva de Usuário, cada uma com suas respectivas operações.
</p>
<br>

## Perspectiva Do Biblioteário
<p>
  O bibliotecário é previamente cadastrado no sistema por um administrador de banco de dados que o fornece suas informações de login para que ele possa utilizar o sistema.
</p>
<br>

## Login
<p>
  Tela de Login, aonde o bibliotecário insere as suas informações de login (CPF e Senha) e entra no sistema.
</p>

</p>
<br>

## Menu Principal
<p>
  Menu Principal do bibliotecário com as opções de <strong>Usuários</strong>, <strong>Livros</strong>, <strong>Empréstimos</strong> e <strong>Sair</strong>, cada opção redirecionando o bibliotecário para um submenu com opçoes relacionadas.<br>
  A opção de <strong>Sair</strong> faz com o que o bibliotecário saia do sistema e volte a tela de login.

## Realizar Empréstimo - Primeira Etapa
<p>
  Tela De Empréstimo, o bibliotecário insere a o CPF, solicita a senha do usuário e o título do livro em questão.
</p>
<p>
  <img src ="https://github.com/CarlosVinicios99/SIstema-Web-Gerenciamento-De-Emprestimos-De-Livros/blob/main/assets/imgs/emprestimo-de-livro.jpg?raw=true" alt="Tela De Empréstimo De Livro">
</p>
<br>

## Realizar Empréstimo - Segunda Etapa
<p>
  Após inserir as informações são exibidos os resultados dos livros, a disponibilidade e o proprietário. <br>
  Se o livro for da biblioteca e estiver disponível o botão de empréstimo é habilitado. <br>
  Caso o livro pertença a algum usuário o e-mail de contato é exibido e disponibilizado para o usuário que deseja o livro emprestado.
</p>
<p>
  <img src ="https://github.com/CarlosVinicios99/SIstema-Web-Gerenciamento-De-Emprestimos-De-Livros/blob/main/assets/imgs/emprestimo-de-livro-2.jpg?raw=true" alt="Tela De Empréstimo De Livro - segunda etapa">
</p>
<br>

## Devolução
